# 윤문 실행 가이드 (rewriting guide)

`ai-tell-taxonomy.md`로 탐지한 AI 티를 **의미는 보존한 채** 자연스럽게 고쳐 쓰는 방법과,
끝난 뒤 스스로 점검하는 체크리스트를 담는다.

## 핵심 원칙

1. **의미 100% 보존** — 고유명사·수치·인용·날짜는 한 글자도 바꾸지 않는다. 주장의 방향·강도·인과·수량도 그대로.
2. **근거 기반 편집** — 탐지된 구간만 손댄다. "왠지 더 좋을 것 같아서" 멀쩡한 문장을 건드리지 않는다.
3. **장르·격식 일관성** — 입력의 장르(칼럼/리포트/블로그/공식문서)와 높임 수준(register)을 유지한다.
4. **과윤문 금지** — 고치는 양이 많아질수록 의미가 흔들린다. 변경은 최소 침습으로.
5. **인공 표현 추가 금지** — AI 티를 없애려다 또 다른 어색한 말투를 새로 만들지 않는다.

## 윤문 강도

사용자 지정(`강도: 보수|기본|적극`)에 따라 손대는 범위를 조절한다. 기본값은 **기본**.

| 강도 | 처리 범위 |
|---|---|
| **보수** | 심각도 **강**만 제거. 구조·어휘는 최대한 보존 |
| **기본** | 강 + 반복되는 **중**(문서당 3회 이상) 제거 |
| **적극** | 강 + 중 전반 + 리듬·구조까지 적극 재배치 (단, 의미 보존은 동일) |

## 편집 순서

먼저 지우면 문장이 짧아져 뒤 작업이 쉬워진다. 아래 순서를 권장한다.

1. **상투어·hedging 삭제**(3) — 자리만 채우는 정형구를 먼저 들어낸다.
2. **번역투 교정**(1) — "~을 통해", 무생물 주어, 직역 대명사 등 구조를 바로잡는다.
3. **형식명사·명사화 풀기**(6) — 명사 덩어리를 동사로 되돌린다.
4. **피동·사동 → 능동**(2)
5. **접속사·연결어 정리**(4) — 문두 접속사·연결어미 뒤 쉼표 정돈.
6. **과잉 수식 덜기**(7) — 군더더기 부사·`-적`·동의어 중첩 축소.
7. **영어 인용 정리**(10)
8. **구조·시각 장식 정돈**(5, 8) — 불릿·헤더·볼드·이모지·대시를 장르에 맞게.
9. **리듬 다듬기**(9) — 종결어미·문장 길이·문두를 변주. 마지막에 전체를 읽으며 손본다.

## 카테고리별 윤문 메모

세부 패턴과 예시는 `ai-tell-taxonomy.md` 각 절을 따른다. 실행 시 유의점만 덧붙인다.

- **번역투**: 직역 대명사는 지시 대상이 분명하면 생략이 가장 자연스럽다. 모호해지면 명사로 복원.
- **피동**: 행위자가 분명한데도 피동이면 능동으로. 행위자를 숨기는 게 의도인 공식 문체는 둔다.
- **상투어**: "결론적으로"를 지운 뒤 문장이 붕 뜨면, 앞 문장과 자연스럽게 잇거나 한 박자 정리어를 살린다.
- **접속사**: 0으로 만들지 않는다. 역접·인과의 핵심 접속사는 남겨 논리를 지킨다.
- **구조**: 절차·사양·체크리스트의 목록은 유지. 짧은 일반 산문의 과한 불릿만 풀어쓴다.
- **명사화**: 전문·법률 표현에서 명사화가 더 정확하면 그대로 둔다.
- **리듬**: 종결을 섞되 격식체 일관성을 깨지 않는다(해요체와 하십시오체를 섞지 않기).

## 자체검증 6항 체크리스트

윤문을 마치면 결과물을 원문과 대조해 아래 6항을 점검한다. 위반 시 해당 edit을 **롤백하고 1회 부분 재작업**한다.

1. **사실 보존** — 고유명사·수치·인용·날짜·단위가 100% 그대로인가?
2. **의미 동등** — 주장 방향·인과·수량·강도가 원문과 같은가? (증가↔감소, 대부분↔일부 등 변질 없음)
3. **변경률 적정** — 과윤문이 아닌가? (대략 본문 30~40% 이내 변경 권장, 50% 초과 시 강제 재검토)
4. **장르·격식 일관** — 장르와 높임 수준(register)이 입력과 같은가?
5. **강 패턴 잔존 0건** — 심각도 '강' 패턴이 결과물에 남아 있지 않은가?
6. **인공 표현 무추가** — 윤문 과정에서 새로 어색한 말투를 만들지 않았는가?

## 변경률·등급 산정

- **변경률** = 대략적인 수정 비율(글자/표현 기준 추정치). 정확한 알고리즘 불요 — 체감 추정으로 충분.
  50%를 넘으면 과윤문 신호 → 보수적으로 되돌린다.
- **등급**:

| 등급 | 기준 |
|---|---|
| **A** | 강 잔존 0, 자체검증 6/6, 변경률 적정, 장르·격식 유지 |
| **B** | 강 잔존 0, 자체검증 5/6 이상, 사소한 중 패턴 일부 잔존 |
| **C** | 강 1건 이하 잔존 또는 변경률 과다 우려 — 재검토 권장 |
| **D** | 강 다수 잔존 또는 의미 훼손 위험 — 부분 재작업 필요 |

등급 B 이하이거나 의미 보존이 불안하면, 사용자에게 한 줄로 알리고 추가 점검(특정 카테고리만 재실행,
강도 하향 등)을 제안한다.
